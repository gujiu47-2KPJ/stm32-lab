# 项目总结：串口收发HEX数据数据包

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA9  | USART1_TX（串口发送） |
| PA10 | USART1_RX（串口接收） |
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| PA0~PA3或PA1 | 按键（切换/触发发送） |
| 3.3V | OLED VCC |
| GND  | OLED GND / 串口GND / 按键GND |

## 项目效果

通过按键发送固定格式的 4 字节 HEX 数据包，同时接收对方发来的数据包，OLED 分别显示收发内容。

## 功能描述

- 初始化 OLED、按键、串口
- 预设4字节发送包 `Serial_TxPacket[4]` 初始值为 0x01~0x04
- 按按键1：发送包每个字节+1，然后调用 `Serial_SendPacket()` 发送，OLED 第2行显示发送的4个字节
- 收到数据包时：OLED 第4行显示收到的4个字节
- 使用自定义数据包协议（包头+数据+校验+包尾）
