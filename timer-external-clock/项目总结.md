# 项目总结：定时器外部时钟

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| TIM_ETR 或 TI 引脚 | 外部脉冲信号输入 |
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| 3.3V | OLED VCC |
| GND  | OLED GND / 信号源GND |

## 项目效果

使用外部信号作为定时器的时钟源（而非内部时钟），测量外部脉冲数量，并在 OLED 上显示。

## 功能描述

- 初始化 OLED 和定时器
- TIM2 的时钟源设置为外部引脚输入（ETR 或 TI 外部触发模式）
- 每检测到一个外部脉冲，定时器计数器+1
- 同时在定时器更新中断中对变量 Num 做累加
- OLED 第1行显示 Num（中断次数），第2行显示定时器当前计数值
- 演示了定时器外部时钟模式
