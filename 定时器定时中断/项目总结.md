# 项目总结：定时器定时中断

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| 3.3V | OLED VCC |
| GND  | OLED GND |

## 项目效果

利用 TIM2 定时器产生定时中断，在中断中累加变量 Num，OLED 实时显示 Num 的变化。

## 功能描述

- 初始化 OLED 和定时器 TIM2
- TIM2 配置为定时中断模式（一定时间间隔触发一次更新事件）
- 每次中断进入 `TIM2_IRQHandler()`，`Num++`
- OLED 第1行实时显示 Num 的数值
- 演示了定时器中断的基本用法
