# 项目总结：bluetooth_OLED_Uitrasonic

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA0  | HC-SR04 TRIG（超声波触发） |
| PA1  | HC-SR04 ECHO（超声波接收） |
| PA2  | 左电机方向A（AIN1） |
| PA3  | 左电机方向B（AIN2） |
| PA6  | 右电机方向A（BIN1） |
| PA7  | 右电机方向B（BIN2） |
| PA9  | USART1_TX → 蓝牙模块RX |
| PA10 | USART1_RX → 蓝牙模块TX |
| PB6  | OLED SCL（硬件I2C1） |
| PB7  | OLED SDA（硬件I2C1） |
| 5V / 3.3V | 电机驱动 / 蓝牙 / HC-SR04 / OLED VCC |
| GND | 公共地 |

## 项目效果

蓝牙遥控小车 + 超声波避障 + OLED 实时显示测距信息。

## 功能描述

- **超声波测距**：HC-SR04 模块测量前方障碍物距离（使用 DWT 时钟周期计时）
- **OLED 显示**：实时显示距离值
- **蓝牙遥控**：通过串口1接收手机指令（f=前进, b=后退, l=左转, r=右转, s=停止）
- **自动避障**：距离<20cm 时强制停车，忽略蓝牙指令（最高优先级）
