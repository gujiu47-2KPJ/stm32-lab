# 项目总结：蓝牙循迹小车

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA9  | USART1_TX → 蓝牙模块RX |
| PA10 | USART1_RX → 蓝牙模块TX |
| PAx（TIMx_CHx） | 左电机PWM |
| PAx（TIMx_CHx） | 右电机PWM |
| GPIO引脚 x4 | 电机方向控制 |
| GPIOB x5 | 5路红外循迹传感器输入 |
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| 5V / 3.3V | 电机驱动 / 蓝牙 / 传感器 / OLED VCC |
| GND | 公共地 |

> 注意：具体电机引脚号取决于 Car.c 和 Motor.c 中的配置。

## 项目效果

计划实现通过蓝牙（串口）遥控 + 循迹传感器自动循迹的小车。

4. 蓝牙遥控优先，循迹作为辅助

## 与"智能小车"项目的区别

"智能小车"是纯循迹版本，"BuleUSCar" 是 HAL 库实现的蓝牙+超声波避障版本。
