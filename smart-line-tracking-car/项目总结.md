# 项目总结：智能小车

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PAx（TIMx_CHx） | 左电机PWM |
| PAx（TIMx_CHx） | 右电机PWM |
| GPIO引脚 x4 | 电机方向控制（AIN1/AIN2/BIN1/BIN2） |
| GPIOB x5 | 5路红外循迹传感器输入 |
| 5V / 3.3V | 电机驱动模块VCC / 传感器VCC |
| GND | 公共地 |

> 注意：具体引脚号取决于 Car.c 和 Motor.c 中的配置，未详细读取。

## 项目效果

使用 5 路红外循迹传感器实现小车自动循迹行驶。

## 功能描述

- 电机驱动：左右两路电机（PWM调速 + 方向控制）
- 5路红外循迹传感器：检测地面黑线的位置
- 根据 5 路传感器的组合状态判断小车位置：
  - 全部为0（无黑线）→ 直行
  - 左偏 → 左转
  - 右偏 → 右转
  - 其他异常 → 停止
- 每 5ms 执行一次循迹决策
