# 项目总结：串口发送+接受

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA9  | USART1_TX（串口发送） |
| PA10 | USART1_RX（串口接收） |
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| 3.3V | OLED VCC |
| GND  | OLED GND / 串口GND |

## 项目效果

通过串口1收发数据，OLED 显示接收到的数据。

## 功能描述

- 初始化 OLED 和串口
- 使用 `Serial_GetRXFlag()` 检测是否有新数据
- 收到数据后：在 OLED 第1行显示该数据的 HEX 值
- 同时通过串口原样返回（回显）
- 相比"串口发送"项目，这里使用了标志位+数据的分离式 API（`GetRXFlag` + `GetRxData`）
