# 项目总结：智能小车

## 项目效果

使用 5 路红外循迹传感器实现小车自动循迹行驶。

## 功能描述

- 电机驱动：左右两路电机（PWM调速 + 方向控制）
- 5路红外循迹传感器：检测地面黑线的位置
- 根据 5 路传感器的组合状态判断小车位置：
  - 全部为0（无黑线）→ 直行
  - 左偏 → 左转
  - 右偏 → 右转
  - 其他异常 → 停止
- 每 5ms 执行一次循迹决策

**注：此版本为纯循迹模式（无蓝牙控制）。**
