# 项目总结：输入捕获模式测频率

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA0  | PWM输出（TIM2_CH1，被测信号源） |
| 输入捕获引脚 | 连接到PWM输出测量频率 |
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| 3.3V | OLED VCC |
| GND  | OLED GND |

## 项目效果

利用定时器输入捕获功能测量 PWM 信号的频率。

## 功能描述

- 初始化 OLED、PWM 作为信号源、IC（输入捕获）
- TIM2 通道1 输出 PWM 信号（作为被测信号源）
- 另一个定时器通过输入捕获测量该信号的频率
- OLED 第1行显示测量到的频率值
- 可设置 PWM 分频系数改变输出频率
