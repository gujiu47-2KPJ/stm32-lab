# 项目总结：BuleUSCar

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PAx  | HC-SR04 TRIG（超声波触发） |
| PAx  | HC-SR04 ECHO（超声波接收 → TIM1输入捕获） |
| PAx（TIM3_CHx） | 左电机PWM |
| PAx（TIM3_CHx） | 右电机PWM |
| GPIO引脚 x4 | 电机方向控制 |
| PA9  | USART1_TX → 蓝牙模块RX |
| PA10 | USART1_RX → 蓝牙模块TX |
| 5V / 3.3V | 电机驱动 / 蓝牙 / HC-SR04 VCC |
| GND | 公共地 |

> 注意：具体引脚号需查看 .ioc 文件确认。

## 项目效果

蓝牙遥控小车 + 超声波自动避障（使用定时器输入捕获测距）。

## 功能描述

- **超声波测距**：HC-SR04 通过 TIM1 输入捕获（上升沿+下降沿）精确测量距离
- **蓝牙遥控**：串口1接收手机指令（F=前进, L=左转, R=右转, S=停止）
- **自动避障**：距离<20cm 时强制停车，不响应蓝牙指令
- 电机控制：左右独立电机，PWM（TIM3）调速

✅ **项目完整，可以跑通**

## 库类型

**HAL 库**（STM32CubeMX 生成）

## 与 bluetooth_OLED_Uitrasonic 的区别

- 使用定时器输入捕获测距（更精确），后者使用 DWT 时钟周期计时
- 没有 OLED 显示
- 使用 HSI 内部时钟（不使用外部晶振）
