# 项目总结：串口发送

## 项目效果

通过串口1（USART1）发送数据，OLED 屏幕显示接收到的数据和串口返回的数据。

## 功能描述

- 初始化 OLED 和串口
- 循环检测是否有串口数据收到
- 收到数据后：在 OLED 上显示该数据的 HEX 值，并通过串口原样返回（回显功能）
- 使用 PA9 (TX) 和 PA10 (RX)

✅ **项目完整，可以跑通**
