# 项目总结：FlashSystemDataProtect

## 项目效果

断电保护按钮计数器 — 按按钮计数并保存到 W25Q64 Flash，断电后数据不丢失。

## 功能描述

- PA1 计数键：按下计数+1，自动保存到 W25Q64 Flash
- PA0 清零键：按下归零，自动保存到 Flash
- SSD1306 OLED（I2C）实时显示当前计数值和上次断电时的值
- 每次上电自动从 Flash 恢复上次断电前的计数

## 完整度

✅ **项目完整，可以跑通**

## 库类型

**HAL 库**（STM32CubeMX 生成）
- SPI1（W25Q64 Flash，Mode 3，4.5Mbps）
- I2C1（SSD1306 OLED，100kHz）

## 数据格式

```
Flash地址0x000000: [0xA5] [count:4字节] [checksum:1字节]  （__packed 紧凑结构体）
```

## 注意

需要外接 W25Q64 模块（CS→PA4, SCK→PA5, MISO→PA6, MOSI→PA7）。
