# 项目总结：智能风口系统

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA0  | 舵机PWM（TIM2_CH1，控制风门0°~180°） |
| PA1  | 电机方向A（AIN1） |
| PA2  | 电机方向B / PWM（AIN2 / TIM2_CH3） |
| PA4  | 电机使能（STBY） |
| PA3  | 紧急停止按钮（输入下拉） |
| PA6  | 绿色LED（TIM3_CH1 PWM呼吸灯） |
| PA7  | 红色LED（普通GPIO） |
| PA9  | USART1_TX（串口通信，115200bps） |
| PA10 | USART1_RX（串口通信） |
| 3.3V | LED正极 / 舵机VCC |
| GND  | 公共地 |

## 项目效果

通过串口发送密码来控制风门开关和电机正反转，配有 LED 状态指示和紧急停止功能。

## 功能描述

这是一个比较完整的智能控制系统：

- **舵机**（PA0, TIM2_CH1）：控制风门打开/关闭（0°~180°）
- **电机**（PA1控制方向, PA2 PWM调速, PA4使能）：正转进风/反转出风
- **LED 指示灯**：PA6 绿色 PWM 呼吸灯（正常运行）、PA7 红色 GPIO 灯（错误/锁机）
- **紧急按钮**（PA3）：按下后紧急停止所有设备
- **串口命令**（USART1, PA9/PA10, 115200bps）：
  - `A密码E` 格式验证密码（默认密码 87654321）
  - 密码正确 → 风门打开 + 电机正转 + 绿灯呼吸效果
  - 密码错误 → 风门关闭 + 电机反转1秒 + 红灯闪烁3次
  - 3次密码错误 → 系统锁机10秒
  - `RECOVER` 恢复命令
  - `TEST` 状态查询

**代码量约 700 行，是工作区最复杂的项目。**
