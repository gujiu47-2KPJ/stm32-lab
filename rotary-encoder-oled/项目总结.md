# 项目总结：roundoled(OLED Dirve)

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA0/PA1（TIMx_CH1/CH2） | 旋转编码器A/B相（编码器模式） |
| 按键引脚 | 编码器按键（归零功能） |
| PB6  | OLED SCL（硬件I2C1） |
| PB7  | OLED SDA（硬件I2C1） |
| 3.3V | OLED VCC / 编码器VCC |
| GND | 公共地 |

## 项目效果

旋转编码器角度显示器 — 通过旋转编码器调节角度值，OLED 实时显示。

## 功能描述

- 旋转编码器（如 EC11）接入定时器编码器模式
- 旋转编码器调整角度值（0°~360° 或其他范围）
- 按钮短按/长按归零
- SSD1306 OLED 显示当前角度
- 版本标注为"按钮归零功能终极修复版"，说明之前有按钮消抖问题的修复过程
- 代码约 800 行，比较完善
