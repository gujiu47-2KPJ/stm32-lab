# 项目总结：stm32led

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PC13 | 板载LED（开漏输出） |
| PA9  | 外接LED1（推挽输出） |
| PA10 | 外接LED2（推挽输出） |
| PA0  | 按键（下降沿中断，上拉输入） |
| GND  | 按键另一端 |

## 项目效果

HAL 库版本的多路 LED 控制（PC13 + PA9 + PA10 + PA0 按键中断）。

## 功能描述

- PC13：开漏输出，控制板载 LED（初始高电平→熄灭）
- PA9、PA10：推挽输出，控制外接 LED（PA9亮, PA10灭）
- PA0：配置为下降沿中断（外部按键触发）
- 使用 HSI 内部时钟

## 完整度

✅ **项目完整，可以跑通**

## 库类型

**HAL 库**（STM32CubeMX 生成）

## 注意

`main.c` 中的 `MX_GPIO_Init` 配置了 PA0 中断但 `stm32f1xx_it.c` 中可能需要补充 `EXTI0_IRQHandler` 的实际处理代码。
