# 项目总结：PWM驱动LED呼吸灯

## 项目效果

使用 PWM 输出控制 LED 亮度，实现呼吸灯效果（渐亮→渐暗→循环）。

## 功能描述

- 初始化 OLED 和 PWM（定时器输出比较模式）
- 主循环中：
  - `for(i=0→100)` 逐渐增加占空比，LED 逐渐变亮（每次延时10ms）
  - `for(i=0→100)` 逐渐减小占空比，LED 逐渐变暗
- `PWM_SetCompare1(i)` 控制 PWM 的占空比
- OLED 可选显示当前值
