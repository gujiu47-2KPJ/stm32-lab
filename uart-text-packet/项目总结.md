# 项目总结：串口收发文本数据包

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA9  | USART1_TX（串口发送） |
| PA10 | USART1_RX（串口接收） |
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| PA0  | 按键 |
| 3.3V | OLED VCC |
| GND  | OLED GND / 串口GND |
