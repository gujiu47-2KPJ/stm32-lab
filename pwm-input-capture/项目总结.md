# 项目总结：PWMI模式测频率占空比

## 接线图

| STM32 引脚 | 连接外设 |
|------------|----------|
| PA0  | PWM输出（TIM2_CH1，被测信号源） |
| PWMI输入引脚 | 连接到PWM输出，同时测频率和占空比 |
| PB8  | OLED SCL（软件I2C） |
| PB9  | OLED SDA（软件I2C） |
| 3.3V | OLED VCC |
| GND  | OLED GND |

## 项目效果

利用定时器 PWMI（PWM输入模式）同时测量 PWM 信号的频率和占空比。

## 功能描述

- 初始化 OLED、PWM（作为信号源）、IC（输入捕获，PWMI模式）
- PWM 信号从某引脚输出，再用另一个定时器的输入捕获通道接收
- PWMI 模式可以同时捕获信号的上升沿和下降沿
- OLED 第1行显示频率，第2行显示占空比（百分比）
- 相比"输入捕获模式测频率"项目，这里增加了占空比的测量
